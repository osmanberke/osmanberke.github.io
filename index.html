AI-based differential diagnosis of dementia etiologies on multimodal data
C Xue, SS Kowshik, D Lteif, S Puducheri, VH Jasodanand, OT Zhou, OB Guney, et al.
Nature Medicine, 2024
A multimodal deep learning framework that differentially diagnoses six dementia etiologies—including Alzheimer's disease and frontotemporal dementia—from clinical notes, neuroimaging, and biomarkers. The model achieves expert-level accuracy while providing interpretable, clinician-aligned predictions.

Interpretability-driven active feature acquisition in learning systems
OB Guney, KS Saichandran, K Elzokm, Z Zhang, VB Kolachalama
International Conference on Machine Learning (ICML), 2025
An active feature acquisition framework that uses explainability-based feature ranking to selectively query informative features at test time, reducing acquisition cost while preserving predictive performance—with applications to neurocognitive disorder recommendation systems.

A deep neural network for SSVEP-based brain-computer interfaces
OB Guney, M Oblokulov, H Ozkan
IEEE Transactions on Biomedical Engineering, 2021
A convolutional neural network tailored for steady-state visually evoked potential (SSVEP) recognition in BCI spellers. The network learns discriminative spectral-spatial features from raw EEG signals, achieving state-of-the-art accuracy across multiple subjects and datasets.
